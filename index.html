CPAN Book

Some time ago I started writing a book together with a friend (Nuno Carvalho), on some notable Perl modules (or frameworks). It will cover different kind of modules, ranging from OO to web frameworks. We have its table of contents mostly defined, but we will uncover it progressively.

The book will be free for download. A printed version will also be available. For now, draft versions of some of the chapters are available for review. Comments are, of course, welcome. The order of the chapters is not final.

  • Chapter 1: The Comprehensive Perl Archive Network [draft]
  • Chapter 2: Modern Object Oriented Perl (Moo, Mouse and Moose) [draft]
  • Chapter 3: Template Systems (Text::Template, HTML::Template::Pro, Template::Toolkit and Text::Xslate)
  • Chapter 4: ...
  • ...

The book is being rendered with XeLaTeX, and is written in Pseudo Pod. The conversion to LaTeX code is performed by chromatic Pod::PseudoPod::LaTeX module. The book is available on GitHub as well.
